## 互動效果

### 滑鼠懸停（Hover）
```
正常狀態:     懸停狀態:
┌─────────┐   ┌─────────┐
│  按鈕   │   │  按鈕   │  ← 顏色變亮
└─────────┘   └─────────┘
```

### 點擊效果
```
點擊後按鈕會閃一下，然後進入下一頁
```


## 顏色方案

```
背景色:          深藍灰色 (20, 20, 30)
標題:            純白色   (255, 255, 255)
文字:            淺灰色   (200, 200, 200)
按鈕（正常）:    藍色     (50, 100, 150)
按鈕（懸停）:    亮藍色   (70, 130, 180)
按鈕（選中）:    綠色     (100, 180, 100)
邊框:            灰色     (100, 100, 120)
```

## 鍵盤操作

```
ESC:   ◀──  返回上一步
       
       或退出程序（在第一頁時）
```
